### 최소힙
- push 과정
    1. 같은 레벨 노드에 원소를 삽입하고 부모와 비교한다
    2. 부모와 비교했을 때 부모가 더 크다면 자리를 바꿔준다.
- pop 과정
    1. 루트노드 값이 나온다
    2. 맨마지막 노드에 값이 루트로 옮겨진다.
    3. 루트 노드의 자식노드중 작은 노드와 값을 비교한다.
    4. 자식이 작다면 자리를 바꿔준다.
- 구현
    - heapq import 하기
    ```
    import heapq as hq
    ```
    - 빈배열을 하나만들고 사용한다.(a=[])
    - pop 사용
    ```
    hq.heappop(a)
    ```
    - push 사용
    ```
    # a에 n을 push
    hq.heappush(a,n)
    ```

### 최대힙
- 루트노드에 가장 큰 원소가 있게된다.
- 구현
    - 최소힙 구현과 비슷하다.
    - pop과push를 할 때 부호만 바꿔주면 최대힙을 구현할 수 있다.
     - pop 사용
    ```
    -hq.heappop(a)
    ```
    - push 사용
    ```
    # a에 n을 push
    hq.heappush(a,-n)
    ```
